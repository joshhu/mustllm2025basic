## 課程簡介
課程內容包含大型語言模型的現況，產品，原理及微調，不包括語言模型的訓練。

## 課程目標
- 了解大型語言模型的產品，現況及應用
- 了解大型語言模型的原理
- 了解大型語言模型的基礎設計
